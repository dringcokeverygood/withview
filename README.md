# 서비스 소개

## 1. 서비스 설명

**개요**

- 한줄소개 : 실시간 사진/영상 공유 화상채팅 어플리케이션
- 서비스명 : **withview**

**타켓**🎯

- 전국으로 흩어진 친구들과 만나서 놀고싶은 사람들
- 기존의 화상채팅 서비스는 지루하다! 새로운 환경에서 화상채팅을 즐기고 싶은 사람들

👉 **즐거움을 추구하는 모든 사람들**
